# Notice_Me_Plz_App
  
## 概要 
　メッセージを送ってもいつも気づいてくれない、「通知オフってたわ」という人に対して、あらゆるサービスでのメール・DMを送るアプリです。
 
  - Gmailでメール 
  - SlackでDM 
  - 決められた時間後に通知を送る「タイマー機能」
  - 決められた時間ごとにメールを再送信する「リピート機能
